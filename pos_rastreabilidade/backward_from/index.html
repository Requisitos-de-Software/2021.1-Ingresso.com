    2. Introdução

    A intenção do documento de Backward-From é ligar todos os requisitos elicitados durante o desenvolvimento do projeto no contexto da matéria aos respectivos métodos de elicitação e modelagem. Para isso, foi utilizada uma matriz de rastreabilidade, onde é possível ver de qual artefato um requisito funcional (RF) ou não funcional (RNF) provêm.
